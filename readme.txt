== Description ==
If you write about somebody at Facebook or Google Plus - mark his name with an icon!

i.e. [userid type="blogspot" id="torvalds-family"]Linus Torvalds[/userid]

The following social networks are supported:

* facebook
* twitter
* google_plus
* Wordpress
* Livejournal
* liruboy / lirugirl
* vk
* habrahabr
* github
